Accueil Maîtrise Savoir-Faire Expérience

Cette page présente en détail mes compétences techniques et ma maîtrise du cycle complet d’un projet web. Du développement front-end " HTML, CSS, JavaScript " à l’intégration back-end " PHP, gestion de bases de données ", je conçois des solutions complètes, ergonomiques et fonctionnelles.
Je suis également en mesure de mettre en place des WORKFLOWS " git push " de déploiement via GitHub, en utilisant les actions GitHub " GitHub Actions " pour automatiser les étapes de build, test, et mise en production et l’intégration en continue " CI/CD " pour garantir performance, fiabilité et évolutivité.
En parallèle, je gère les projets selon les principes de LA METHODE AGILE avec l'application " TRELLO " : planification par sprint, suivi des tâches, réunions d’équipe et adaptation continue sont au cœur de ma méthode de travail.

Je crois en l’importance d’un code sémantique, bien structuré, maintenable et évolutif, facilitant la lisibilité, la collaboration et l’optimisation sur le long terme. Que ce soit en front-end ou en back-end, j’aime relever les défis techniques. La collaboration en équipe est aussi un moteur dans mes projets.

Mes compétences clés en développement Web

▼

HTML (HyperText Markup Language) définit la structure d’une page web via des balises. Il organise titres, paragraphes, listes, tableaux ou formulaires pour le navigateur. IL intègre audio, vidéo et éléments sémantiques pour un code optimisé.

CSS (Cascading Style Sheets) gère la présentation des pages web. Il définit couleurs, polices, espacements et mises en page pour un design harmonieux. Avec Flexbox et Grid, CSS permet des mises en page modernes et responsives.

JavaScript est le langage qui rend les pages web interactives et dynamiques. Il permet de manipuler le DOM, gérer des événements ou communiquer avec des APIs. Grâce aux frameworks modernes, il est incontournable dans le développement web.

PHP est un langage de script côté serveur utilisé pour créer des sites web dynamiques. Il interagit avec les bases de données, génère du HTML et gère les formulaires. PHP reste très présent dans le web grâce à sa simplicité et à des CMS comme WordPress.

phpMyAdmin est un outil web qui permet de gérer facilement des bases de données MySQL. Il offre une interface graphique pour exécuter des requêtes, créer des tables ou importer des données. C’est un outil incontournable pour administrer ses bases sans écrire toutes les commandes SQL.

WordPress est un système de gestion de contenu (CMS) très populaire pour créer des sites web. Il permet de concevoir des blogs, des vitrines ou des boutiques en ligne sans coder. Grâce à ses thèmes et plugins, WordPress offre flexibilité et personnalisation à tous les projets.

GitHub est une plateforme de collaboration et de gestion de versions pour les développeurs. Elle permet d’héberger du code, suivre les modifications et travailler en équipe grâce à Git. Avec ses fonctionnalités comme les pull requests, GitHub facilite le développement collaboratif.

Trello est un outil de gestion de projet en ligne . Il fonctionne sous forme de tableaux où l’on peut créer des listes et des cartes pour organiser des tâches, des idées ou des projets. Grâce à son interface visuelle, il permet de suivre facilement l’avancement du travail, seul ou en équipe. C’est idéal pour planifier des projets, collaborer à distance et rester organisé au quotidien.
